nl.sara.webdav.Client(host, useHTTPS, port)
Parameters:
{String} host Optional
Optional; The hostname or IP address of the server. This is only needed if the host is different from the one serving this library.
{Boolean} useHTTPS Optional, Default: false
Optional; If set to true, HTTPS is used. If set to false or omitted, HTTP is used. This parameter is ignored if host is not set.
{Number} port Optional
Optional; Set a custom port to connect to. If not set, the default port will be used (80 for HTTP and 443 for HTTPS). This parameter is ignored if host is not set.

Perform a WebDAV COPY request
Parameters:

The path to perform COPY on
{Function(status|Multistatus)} callback
Querying the server is done asynchronously, this callback function is called when the results are in
{String} destination
The destination to copy to. Should be either a full URL or a path from the root on this server (i.e. it should start with a /)

Optional; Specify what to do when destination resource already exists. Should be either FAIL_ON_OVERWRITE or SILENT_OVERWRITE. The default is SILENT_OVERWRITE.

Optional; Should be '0' or 'infinity'. This is used in case of a collection; 0 means only copy the collection itself, infinity means copy also everything contained in the collection

Perform a WebDAV PROPFIND request
Parameters:

The path get a PROPFIND for

Querying the server is done asynchronously, this callback function is called when the results are in

Optional; Value for the 'depth' header, should be either '0', '1' or the class constant INFINITY. When omitted, '0' is used. See RFC 4916.

Optional; The properties to fetch. Should be either one of the class constants PROPNAME or ALLPROP or an array with Property objects. When omitted, ALLPROP is assumed. See RFC 4916.

Optional; An array with Property objects used for the element. This is only used for ALLPROP requests. When omitted, no element is send. See RFC 4916.
